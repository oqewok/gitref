git add ключи

Описание

Добавляет изменения в индекс

Ключи

.
Добавляет все измененные файлы.
папка
Включает все изменения внутри папок и сами папки.
файл
Включает все изменения файла.
–u
Включает все изменения уже отслеживаемых файлов.

git commit –m сообщение

Описание

Объединяет все изменения в отдельный коммит и дает ему соответствующее сообщение, данное в качестве параметра ключа.

git commit –am сообщение

Описание

Добавляет все изменения в отслеживаемых файлах из рабочей директории в отдельный коммит с переданным в качестве параметра ключа сообщением.

git reset HEAD ключи

Описание

Удаляет изменения из индекса и хранит их только в рабочей директории.

Можно передать те же ключи в команду git add для выбора тех изменений, которые нужно удалить.

git reset ––soft HEAD~

Описание

Отменяет последний коммит. Изменения все еще будут храниться в индексе.

git diff

Описание

Показывает изменения всех отслеживаемых файлах в рабочей директории.

git diff ––cached

Описание

Показывает изменения файлов в индексе.

git log ключи

Описание

Показывает историю коммитов.

Ключи

––oneline
Вывод в сокращенном формате, показываются только сокращенных хэш и заголовок.
–p
Также показывается разница в измененных файлах.
––stat
Также показывается статистика по количеству удалений и добавлений к каждому коммиту.

git status ключи

Описание

Показывает имена файлов с изменениями в рабочей директории и индексе.

Ключи

–s
Вывод в сокращенном формате.
